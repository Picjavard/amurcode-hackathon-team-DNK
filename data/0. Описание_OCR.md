Описание данных для использования при решении кейсового задания «Интеллектуальны отбор данных. Конструктор аналитических выборок для бюджетного планирования»
В папке «РЧБ» содержатся бюджетные данные по месяцам нарастающим итогом в разрезе бюджетов: областного и муниципальных образований области. Наименования полей содержатся в файлах выгрузки:
— Доведенные лимиты -поле «лимиты ПБС 2025 год»;
— Принятые бюджетные обязательства (заключенные договора /контракты/соглашения) — поле «Подтверждено лимитов по БО 2025 год»;
— Кассовые выплаты — поле «Всего выбытий (бух.учёт)».
В папке «Соглашения» содержится файл с описанием состава выгружаемых данных «Состав выгружаемых данных _З_2.xlsx»:
— данные по соглашениям, заключаемым с муниципальными образованиями области о предоставлении им межбюджетных трансфертов — MBT.csv;
— данные по соглашениям, заключаемым с бюджетными и автономными учреждениями о предоставлении им субсидии на иные цели — ИЦ.csv;
— данные по соглашениям, заключаемым с юридическими лицами, физическими — лицами, индивидуальными предпринимателями о предоставлении им субсидии — ЮЛ_ИП_ФЛ.csv.
В папке «ГЗ» содержится файл с описанием состава выгружаемых данных «Состав выгружаемых данных _ГЗ.xlsx»:
— Данные с бюджетной классификацией контрактов /договоров - «Бюджетные строки.csv»;
— Данные с реквизитным составом контрактов /договоров - «Контракты и договора.csv»;
— Данные с фактами оплаты контрактов /договоров - «Платежки.csv».
СКК — контрольный пример. Красным отмечены поля из выгрузки соглашений.
